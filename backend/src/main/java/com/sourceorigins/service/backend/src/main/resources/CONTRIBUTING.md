# Guide de contribution — Source Origin

## Règles du groupe
1. Chaque membre travaille sur sa propre branche
2. Messages de commit clairs et descriptifs
3. Toute Pull Request approuvée avant merge
4. Ne jamais pusher directement sur main

## Format des commits
- feat: nouvelle fonctionnalité
- fix: correction de bug
- docs: documentation
